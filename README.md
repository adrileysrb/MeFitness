# MeFitness
 Aplicativo para gerenciamento de treinos.
 
Tecnologias:
<ul>
  <li>Android Studio</li>
  <li>Java</li>
  <li>Firebase Firestore</li>
  <li>Firebase Authentication</li>
  <li>Firebase Storage</li>
  <li>MaterialShowcaseView</li>
  <li>Glide</li>
 
</ul>
